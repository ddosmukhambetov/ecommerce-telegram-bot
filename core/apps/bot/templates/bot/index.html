Ecommerce Telegram Bot

Project Description

  • This is a simple Telegram bot page built with Django and Aiogram.
  • Using Django Admin, you can add and remove categories and products. PostgreSQL is used as the database.
  • Includes a user registration, login, and password recovery system.
  • In the Telegram bot, users can browse categories and view products.
  • The bot provides commands like /help, /description, /catalog, and /admin.
  • The /admin command opens the admin menu, which includes exclusive features such as "broadcast" and more.
  • The Django Admin panel is fully configured for easy management.

Click here to access Django Admin, or add /admin/ to the end of the URL.
